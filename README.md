# Alex-IO

Complete protective packaging quoting platform for foam fabricators and packaging shops. Quotes foam inserts + corrugated RSC/mailer cartons together in one step.

Live: https://api.alex-io.com

## Stack
- Next.js 16, Tailwind v4
- PostgreSQL on Render (raw pg, no ORM)
- Microsoft 365 via MS Graph API for email
- OpenAI GPT-4.1-mini for AI orchestration
- Upstash Redis for rate limiting
- Cloudflare DNS, Render hosting
